=======
django-email
=======
:Info:  See `github <https://github.com/rahulagarwal86/django-email>`_ for the latest source.
:Author: Rahul Agarwal
:Maintainer: Rahul Agarwal<agarwalrahul.1986@gmail.com>

About
=====

Django-Email is use to send email by dynamically changing content in Database.
It is easy to use and easily maintainable.

Requirement
===========

1.Email host server 
2. BCC e-mail address

Issues / Questions / Feedback
=============================

Any issues with, questions about, or feedback for django-email should be
sent to the maintainer email.Issue or feedback will be addressed as early as possible.

Installation
============

Add mailer in INSTALLED_APPS of settings file.
python manage.py syncdb

How to use
===========

How to use email is defined in example app

Dependencies
============

The PyMongo distribution is supported and tested on Python 2.x, where
x >= 4 and django 1.2 or greater
